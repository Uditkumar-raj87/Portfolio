# Udit Kumar Raj - Portfolio

Personal portfolio website with a dark, cinematic UI, interactive cards, and a modern layout designed for frontend and data-focused work.

## Highlights
- Cinematic 3D background with subtle depth, fog, and floating elements
- Smooth parallax and premium micro-interactions on cards
- Light/Dark theme toggle
- Fully responsive layout

## Sections Included
- Hero introduction
- Internship experience
- Projects
- UI/UX case studies
- Technical skills (icons + summaries)
- Education
- Certificates

## Built With
- HTML
- CSS
- JavaScript

## Run Locally

https://uditkumar-raj87.github.io/Portfolio/  


Open index.html directly, or start a local server:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000

## Customize
- Update content in index.html
- Tweak colors, animations, and layout in styles.css
- Adjust interactions in script.js
