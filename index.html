Elevator Control Panel HCI Analysis

Research board layout with structured insights and concept exploration.

A structured HCI case study that maps real-world elevator usage patterns into clear, accessibility-first UI concepts for smart buildings.

Problem Definition

Traditional elevator panels are confusing, inefficient, and not optimized for modern high-rise usage.

User Pain Points

Difficulty selecting floors quickly, accessibility challenges, unclear feedback, overcrowding inefficiency.

Practical Real-World Issues

Peak hour delays, emergency usability, maintenance limitations, high traffic building usage.

Safety, Usability & Accessibility

Need for tactile feedback, voice assistance, clear indicators, error prevention.

Proposed Smart Control Solutions

AI-assisted floor prediction, touchless controls, voice commands, smart grouping, real-time occupancy logic.

Concept Translation

Low-fidelity user flows, interaction mapping, and concept sketches evolved into high-fidelity smart panel UI ideas.
